## ⭐ 프로젝트 소개

---

- **Who’s Breeder?**
    
    **우수한 견종보존을 위해 동물보호 의식과 견종표준의 이해를 기반으로,**
    
    **견종의 짝짓기, 출산, 질병, 위생 자견분양 등 번식에 전문적인 지식을 갖추어**
    
    **강아지나 고양이의 혈통을 유지하고 올바르게 번식해 애견문화 발전에 기여**
    
- Breeder와 Adopter(분양희망자)를 연결하여, 강아지를 분양받도록 도와주는 프로젝트
